一、Windows 设置 Rust 镜像（PowerShell 推荐）
1 临时配置（当前窗口有效）
PowerShell：
$env:RUSTUP_DIST_SERVER="https://mirrors.tuna.tsinghua.edu.cn/rustup"
$env:RUSTUP_UPDATE_ROOT="https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup"
CMD：
set RUSTUP_DIST_SERVER=https://mirrors.tuna.tsinghua.edu.cn/rustup
set RUSTUP_UPDATE_ROOT=https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup
二、永久配置（推荐）
方法1：PowerShell 设置用户环境变量
[Environment]::SetEnvironmentVariable(
"RUSTUP_DIST_SERVER",
"https://mirrors.tuna.tsinghua.edu.cn/rustup",
"User"
)

[Environment]::SetEnvironmentVariable(
"RUSTUP_UPDATE_ROOT",
"https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup",
"User"
)
关闭终端重新打开即可生效。测试 $env:RUSTUP_DIST_SERVER 是否设置好
打开 PowerShell 执行：
irm https://sh.rustup.rs -UseBasicParsing | iex
或者下载官方安装器：
https://static.rust-lang.org/rustup/dist/x86_64-pc-windows-msvc/rustup-init.exe （双击安装）
Windows Rust 有两个 toolchain：
| Toolchain | 说明                      |
| --------- | ----------------------- |
| msvc      | 使用 Microsoft Visual C++ |
| gnu       | 使用 MinGW                |
推荐：x86_64-pc-windows-msvc


二、Linux和Mac 设置 Rust
配置环境变量：
export RUSTUP_DIST_SERVER=https://mirrors.tuna.tsinghua.edu.cn/rustup
export RUSTUP_UPDATE_ROOT=https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup
如果是 永久配置
echo 'export RUSTUP_DIST_SERVER=https://mirrors.tuna.tsinghua.edu.cn/rustup' >> ~/.bashrc
echo 'export RUSTUP_UPDATE_ROOT=https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup' >> ~/.bashrc
source ~/.bashrc
如果是 zsh
echo 'export RUSTUP_DIST_SERVER=https://mirrors.tuna.tsinghua.edu.cn/rustup' >> ~/.zshrc
echo 'export RUSTUP_UPDATE_ROOT=https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup' >> ~/.zshrc
source ~/.zshrc
安装 Rust
执行官方安装脚本：
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
选择：1) Proceed with installation
安装完成后执行：source $HOME/.cargo/env

配置 Cargo 国内源（极其重要）
Rust 依赖默认从 crates.io 下载，在中国会很慢。
推荐使用 清华镜像。
Linux和Mac
创建配置文件：
mkdir -p ~/.cargo
vim ~/.cargo/config.toml
内容：
[source.crates-io]
replace-with = 'tuna'

[source.tuna]
registry = "sparse+https://mirrors.tuna.tsinghua.edu.cn/crates.io-index/"

[net]
git-fetch-with-cli = true


Window·PowerShell：
创建配置目录： mkdir $env:USERPROFILE\.cargo （一般默认创建好了，可能会报目录以及存在）
编辑文件：notepad $env:USERPROFILE\.cargo\config.toml
写入：
[source.crates-io]
replace-with = 'tuna'

[source.tuna]
registry = "sparse+https://mirrors.tuna.tsinghua.edu.cn/crates.io-index/"

[net]
git-fetch-with-cli = true

验证：type $env:USERPROFILE\.cargo\config.toml

四、验证安装
rustc --version
cargo --version
查看版本：rustup show
更新 Rust：rustup update
安装 nightly：rustup install nightly
切换版本：rustup default nightly
卸载 Rust：rustup self uninstall

Rust 常用开发工具（默认就ini时候安装好了）
rustup component add rustfmt  代码格式化  ，使用 cargo fmt
rustup component add clippy   代码质量检查，使用 cargo clippy

测试 Cargo 是否使用镜像
cargo new test_rust
cd test_rust
cargo run
如果依赖下载速度很快，说明镜像已经生效。
